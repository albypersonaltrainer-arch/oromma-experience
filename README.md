# OROMMA Experience

Landing estática premium para OROMMA Experience Germany.

## Cómo subir a GitHub

1. Crea un repositorio llamado `oromma-experience`.
2. Sube todo el contenido de esta carpeta al repositorio.
3. No subas la carpeta comprimida, sube los archivos: `index.html`, `README.md` y la carpeta `assets`.

## Cómo desplegar en Vercel

1. Entra en Vercel.
2. New Project.
3. Importa el repositorio `oromma-experience`.
4. Framework Preset: Other.
5. Build Command: dejar vacío.
6. Output Directory: dejar vacío.
7. Deploy.
